Buttons

There are three different style buttons on an <a>, <button>, or <input> element.

{Core Components}

Example

Button Normal

Link Button

Button Small

Link Button 

//Normal buttons
<button class="rui-button">Cancel</button>
<button class="rui-button rui-button-primary">Save</button>
<button class="rui-button rui-button-danger">Delete</button>
<a href="#" class="rui-button">Link Button</a>

//Small buttons
<button class="rui-button rui-button-sm">Cancel</button>
<button class="rui-button rui-button-primary rui-button-sm">Save</button>
<button class="rui-button rui-button-danger rui-button-sm">Delete</button>
<a href="#" class="rui-button rui-button-sm">Link Button</a>
